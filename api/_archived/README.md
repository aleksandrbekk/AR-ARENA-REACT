# Архив старых скриптов

Эти файлы были архивированы 2026-01-24 при рефакторинге API.

## Fix-скрипты (одноразовые исправления)

Скрипты для ручного исправления данных в базе. Выполнены и больше не нужны.

| Файл | Назначение |
|------|------------|
| `fix-4000.js` | Исправление платежа на 4000 |
| `fix-all-crypto.js` | Массовое исправление крипто-платежей |
| `fix-all-lava.js` | Массовое исправление Lava платежей |
| `fix-crypto-amounts.js` | Корректировка сумм крипто-платежей |
| `fix-demon.js` | Исправление платежа пользователя demon |
| `fix-gross.js` | Исправление gross/net сумм |
| `fix-round.js` | Исправление округления |
| `fix-sergio.js` | Исправление платежа пользователя sergio |

## Check-скрипты (диагностика)

Скрипты для проверки данных. Использовались для отладки.

| Файл | Назначение |
|------|------------|
| `check-client.js` | Проверка клиента |
| `check-crypto.js` | Проверка крипто-платежей |
| `check-manual.js` | Ручная проверка |
| `check-payments.js` | Проверка всех платежей |
| `check-samples.js` | Проверка примеров |

## Не архивировано

- `check-subscription.js` — используется в продакшене для проверки подписок
